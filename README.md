# Houston::Roadmap

A module for Houston to facilitate managing milestones.


## Installation

In your [Houston](https://github.com/houstonmc/houston) `config/config.rb` file, add:

    use :roadmap

And then execute:

    $ bundle

Run Houston and you can navigate to your roadmap at the path '/roadmap'
